# AIP-C01 Practice Exam

Interactive practice exam for the **AWS Certified Generative AI Developer Professional (AIP-C01)** certification.

🔗 **Live exam**: https://joyson-fernandes.github.io/aip-c01-practice-exam

---

## Features

- 75 questions across all five exam domains
- Instant feedback — select an answer and immediately see if you're right or wrong
- Full explanations — why the correct answer is right and why each wrong answer is wrong
- Per-domain performance breakdown in results
- 170-minute countdown timer
- Scaled score out of 1000 (passing score: 720)

## Domain Coverage

| Domain | Weight | Questions |
|--------|--------|----------|
| Foundation Model Integration & Data Management | 31% | 23 |
| GenAI Application Implementation & Integration | 26% | 20 |
| AI Safety, Security & Governance | 20% | 15 |
| Operational Excellence & Efficiency | 12% | 9 |
| Testing, Validation & Troubleshooting | 11% | 8 |
